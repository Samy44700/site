Caractères ASCII et Hexadécimaux

a) Lettres de A à Z et a à z

  1. Majuscules : A - Z : 41 - 5A (Hex)
  2. Minuscules : a - z : 61 - 7A (Hex)

b) Autres caractères

  1. & : 26 (Hex)
  2. " : 22 (Hex)
  3. ’ : C1 (UTF-8)
  4. ( : 28 (Hex)
  5. - : 2D (Hex)
  6. _ : 5F (Hex)
  7. @ : 40 (Hex)
  8. ] : 5D (Hex)

c) Caractères accentués

  1. é : E9 (ISO-8859-1), C3 A9 (UTF-8)
  2. è : E8 (ISO-8859-1), C3 A8 (UTF-8)
  3. ç : E7 (ISO-8859-1), C3 A7 (UTF-8)
  4. à : E0 (ISO-8859-1), C3 A0 (UTF-8)

d) Retours à la ligne

  1. Retour à la ligne : 0A (Hex)
  2. Retour chariot : 0D (Hex)
  3. Sous Windows : 0D 0A

e) Encodage dans Notepad++

  1. ANSI : ISO-8859-1
  2. UTF-8 : Trousseau de tous les caractères Unicode
  3. UTF-8 sans BOM : Semblable à UTF-8 sans le Byte Order Mark
  4. UTF-16 : Gère Unicode mais consomme plus d'espace

Conclusion : Les différents encodages affectent le stockage et l'affichage des caractères. L'éditeur hexadécimal peut être utilisé pour inspecter les valeurs hexadécimales.
